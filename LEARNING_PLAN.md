# Learning Plan

## Goal

Prepare for Junior/Student Java backend interviews by building a Warehouse Management System step by step.

## Current Step

Create a `Product` class in package:

```text
com.sander.warehouse
```

The class should have:

- private fields
- constructor
- getters and setters
- `toString()`

Fields:

- `id`
- `name`
- `quantity`
- `price`

## Topics To Cover Later

- OOP
- encapsulation
- constructors
- collections
- `HashMap`
- `equals()` and `hashCode()`
- design patterns
- singleton
- REST API
- Spring Boot
- SQL
- Docker

## Suggested Workflow

1. Write code manually.
2. Run Maven compile.
3. Read compiler errors.
4. Fix one error at a time.
5. Ask the AI Agent for review, not instant full solution.
6. Commit working steps.
